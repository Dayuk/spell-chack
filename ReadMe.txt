Kmode = 키보드 입력모드
Gmode = 게임중 최소화 모드

기본 사용 방식
1.프로그램 실행 후 로딩중 스펠 카테고리 지정
2.Gmode 실행
3.각 스펠에 맞는 버튼을 누르며 게임 진행

키보드 사용 방식
1.프로그램 실행 후 로딩중 스펠 카테고리 지정
2.Gmode 실행
3.Kmode 실행
4.아래 키를 이용하여 게임 진행(버튼입력 가능)

기본 키보드 입력 키

1, 1  	1, 2  	1, 3  	1, 4  	1, 5
 p      	 o    	  i      	 u      	 y

2, 1  	2, 2  	2, 3  	2, 4  	2, 5
  l      	 k    	  j     	  h      	 g

키보드 입력 키 변경은 Change Input Key버튼으로 가능
각 입력 밑에있는 버튼은 개별저장, Save ALL 버튼은 전체저장
전체저장시 입력돼지 않은 키는 작동하지 않음
